The aim of this work is to visually compare different treemap layouting algorithms with respect to how easy it is to detect structural changes that occur over time. This is done by adding color-coding to the individual rectangles in the treemap.

Components accessible through the navbar:
  • Treemap layout using the AgCharts framework
  • Treemap layout using the amcharts framework
  • Treemap layout using the d3.js framework
